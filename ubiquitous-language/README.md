# Ubiquitous Language

本目录用于维护 MOM 智能问数项目统一术语，减少需求规格、数据模型、接口文档、原型和测试材料之间的表述偏差。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `CONTEXT.md` | 统一术语表，记录项目内推荐使用的领域术语、定义和应避免的说法。 |

## 维护约定

- 出现新术语、同义词冲突或口径争议时，先确认标准说法，再更新 `CONTEXT.md`。
- 术语表只记录项目特有领域概念，不记录通用技术名词和临时实现细节。
- 需求、数据模型、接口、原型和测试材料中的正式表述应优先对齐 `CONTEXT.md`。
