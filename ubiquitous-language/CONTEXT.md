# MOM智能问数统一术语

本文档用于维护 MOM 智能问数项目在需求规格、数据模型、接口文档、原型和测试材料中的统一表述口径。后续出现新术语或口径争议时，先确认标准术语，再写入本文档。

## Language

**默认 Agent**:
MVP 阶段唯一的问数智能体，管理侧配置和用户侧问数均归属于该对象。
_Avoid_: 智能体实例、多 Agent

**管理侧**:
用于维护默认 Agent 配置能力的管理入口，包括模型、数据源、知识、语义模型、Skill、预设问题和访问 API。
_Avoid_: 后台、配置端、管理后台

**用户侧**:
用户发起问数、查看结果和管理会话的运行入口，不维护模型、数据源、知识、语义模型、Skill 和访问 API 配置。
_Avoid_: Chat 端、问答端、前台

**用户侧运行页**:
用户侧承载文本问数、语音问数、预设问题、会话展示和问数结果展示的页面。
_Avoid_: 用户侧 Chat 页面、Chat 页面、问答页

**配置完整可用（配置生效）**:
默认 Agent 已具备运行所需的模型、语音识别模型、数据源、Skill、智能体知识、业务知识、语义模型和预设问题等必要条件，用户侧可进入正式问数。
_Avoid_: 配置完成、配置好了、已配置

**模型资源配置**:
管理侧维护默认 Agent 依赖的大语言模型、嵌入模型和语音识别模型。
_Avoid_: 模型资源、模型管理

**语音识别模型**:
模型资源配置中用于把用户语音输入识别为文本的模型，MVP 阶段属于必配能力。
_Avoid_: 语音模型、语音能力

**数据源配置**:
管理侧维护默认 Agent 可连接的 MOM 业务数据源，支撑 Schema 初始化和问数取数。
_Avoid_: 数据库配置、数据连接配置

**Schema 初始化**:
从数据源读取表、字段等结构信息，形成问数可使用的表结构基础信息。
_Avoid_: 表结构同步、数据库初始化

**数据表范围配置**:
管理侧定义数据源下哪些表纳入默认 Agent 问数范围，哪些表排除，哪些表标记为核心表。
_Avoid_: 表权限配置、数据库授权范围

**核心表**:
问数场景优先关注的业务表，作为表选择、语义配置和测试验证的重点对象。
_Avoid_: 主表、重点表

**排除表**:
明确不纳入默认 Agent 问数范围的表。
_Avoid_: 禁用表、隐藏表

**Skill 配置**:
管理侧维护默认 Agent 可调用的预置工具能力，只配置可用性、调用说明和测试结果。
_Avoid_: 工具映射、自定义脚本配置、自定义工作流配置

**系统提示词**:
默认 Agent 运行时内置的全局提示词，不作为管理侧或用户侧可编辑配置。
_Avoid_: 提示词配置页、自定义提示词、用户提示词配置

**预设问题**:
管理侧维护的问数引导问题。用户侧展示时可称为推荐问题，但数据对象和配置名称统一叫预设问题。
_Avoid_: 推荐问题配置、高频问题、快捷问题

**访问 API**:
管理侧独立入口，用于管理默认 Agent 的外部访问凭证和调用说明。用户侧不维护访问 API。
_Avoid_: 用户侧 API、智能体配置 API、外部调用入口配置

**API 访问配置**:
访问 API 页面维护的外部调用配置对象，包括单个有效 API Key、启停状态和调用说明。
_Avoid_: 用户侧 API 配置、接口配置页

**外部访问 API Key**:
外部系统调用默认 Agent 时使用的访问凭证，完整 Key 仅在生成或重置成功后展示一次。
_Avoid_: 模型 API Key、用户登录凭证

**智能体知识**:
用于增强默认 Agent 问答能力的文档、问答对和 FAQ 类知识。
_Avoid_: 知识库、文档库、Agent 知识库

**业务知识**:
用于维护 MOM 业务术语、状态枚举、业务规则、不可回答边界和业务口径的知识对象。
_Avoid_: 术语库、业务词典、业务规则库

**语义模型**:
用于维护业务对象、字段别名、表关系、指标口径和枚举映射的结构化语义配置。
_Avoid_: 语义知识、语义库、RAG 知识

**语义字段映射**:
语义模型中的具体映射条目，用于描述数据表字段与业务名称、同义词、业务描述等内容的对应关系。
_Avoid_: 语义知识条目、字段知识

**用户会话**:
用户与默认 Agent 的一次连续问答上下文，支持新建、切换、修改标题、置顶和删除。
_Avoid_: 聊天记录、历史对话、会话栏

**会话展示**:
用户侧按时间顺序展示用户消息、语音识别文本、Agent 回复、问数结果和运行提示的窗口区域。
_Avoid_: Chat 窗口、对话展示

**会话消息**:
会话中的单条消息，包括用户输入、语音识别文本、Agent 回复和系统提示。
_Avoid_: 聊天消息、问答内容、对话内容

**问数结果**:
默认 Agent 对用户问题返回的结果对象，可表现为文本、表格、指标卡、图表、澄清提示或边界提示。
_Avoid_: 答案、回复、查询结果

**表格结果**:
用于展示明细清单类问数结果的结果形态。
_Avoid_: 列表结果、明细结果

**指标卡结果**:
用于展示关键数值类问数结果的结果形态。
_Avoid_: 指标结果、数字卡片

**图表结果**:
用于展示图形化问数结果的结果形态，MVP 阶段包括饼图、折线图和柱状图。
_Avoid_: 可视化结果、图形结果

**澄清提示**:
默认 Agent 无法确定业务对象、时间范围、筛选条件或统计口径时，要求用户补充条件的提示。
_Avoid_: 错误提示、失败提示

**边界提示**:
用户问题超出 MVP 问数范围、配置能力或可回答边界时返回的可理解提示。
_Avoid_: 报错、拒答提示

**运行提示**:
用户侧用于展示无数据、无法识别、配置不可用、语音识别失败或服务异常等情况的提示信息。
_Avoid_: 错误提示、异常提示、失败提示

**语音问数**:
用户通过语音输入发起问数，系统识别为文本后进入默认 Agent 问数链路。
_Avoid_: 只用“语音识别”代指完整问数闭环、语音查询

**用户鉴权管理**:
用户登录注册、身份校验、角色权限等面向用户访问控制的管理能力，MVP 阶段不单独建设。
_Avoid_: 用户侧访问范围、API Key 鉴权
