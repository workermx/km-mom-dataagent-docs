# MOM智能问数-MVP用户侧需求规格文档

### 1.1 需求背景

MOM 智能问数 MVP 面向 MOM 业务提供对话式问数入口，MVP 阶段支持工单、库存、设备三类主题。用户在运行页发起文本问题，或通过语音输入识别后手动发送文本问题，由系统基于默认 Agent 完成意图识别、知识增强、业务问数和结果展示。MVP 阶段以管理侧最新需求为准，用户侧不选择 Agent，系统提示词由系统内置，不对用户开放修改。语音输入属于 MVP 必做能力，但仅作为输入辅助。

### 1.2 需求目标

| 目标 | 说明 |
|---|---|
| 支持默认 Agent 问数闭环 | 管理侧完成基础配置后，用户侧可直接发起问数。 |
| 支持文本和语音输入 | 用户可直接输入文字，也可通过后端 ASR 代理识别语音后确认发送。 |
| 支持预设问题引导 | 首页提供高频问题入口，降低提问成本；预设问题在用户侧展示时可称为推荐问题。 |
| 支持会话留痕 | 问答过程可在历史会话中回看。 |
| 支持配置完整可用后运行 | 用户侧只消费配置完整可用的默认 Agent 能力（配置生效）。 |

### 1.3 适用角色

| 角色 | 使用目标 |
|---|---|
| MOM 业务用户 | 新建会话、发起问数、查看结果、查看历史会话。 |
| 内部管理员 | 配置默认 Agent 的模型、数据源、知识、语义、Skill 和预设问题。 |
| 实施人员 | 根据客户数据和业务口径完成默认 Agent 初始化和验证。 |
| 测试人员 | 验证默认 Agent 的问数效果和语音输入效果。 |

### 1.4 MVP 范围

- 默认 Agent 用户侧运行入口
- 新建会话、历史会话查看和会话切换
- 单个历史会话管理，包括修改会话标题、置顶会话和删除会话
- 会话窗口展示，包括用户消息、Agent 回复、语音识别后的文本、问数结果和运行提示
- 当前问数暂停，包括用户在流式问数过程中中止当前回答
- 预设问题（用户侧展示为推荐问题）展示与快捷问数
- 文本问数闭环
- 语音识别后的文本问数闭环
- 问数结果展示，包括文本、表格、指标卡、饼图、折线图和柱状图
- 配置不可用、无数据、无法识别、语音输入不可用等运行提示

用户侧不维护模型、数据源、Skill、智能体知识、业务知识、语义模型和预设问题配置。上述配置由管理侧维护，用户侧只消费配置完整可用的默认 Agent 能力（配置生效）。

## 2. 需求描述

### 2.1 业务描述

MVP 阶段采用 “一个默认 Agent + 管理侧配置 + 用户侧问答” 的方式。管理侧完成默认 Agent 的基础配置后，用户侧在运行页发起文本问题，或通过后端 ASR 代理识别语音后确认发送文本问题，系统返回可读的统计、列表或分析结果，并保留会话留痕。预设问题用于引导用户发起问数，用户侧展示时可称为推荐问题。系统提示词由系统内置，不开放用户修改。

```mermaid
flowchart LR
  A["管理员完成默认 Agent 配置"] --> B["用户进入运行页"]
  B --> C["输入文本 / 语音识别文本"]
  C --> D["默认 Agent 问数能力"]
  D --> E["返回结果 / 说明"]
  E --> F["历史会话留痕管理"]
```

### 2.2 需求边界及验收条件

**包含范围**

- 新建会话、历史会话展示、会话标题与时间展示。
- 每个历史会话支持单独管理，包括修改会话标题、置顶会话和删除会话。
- 预设问题（用户侧展示为推荐问题）卡片：工单状态统计、低于安全库存物料查询、设备停机时长分析。
- 文本输入、语音输入入口、消息发送。
- 基于用户问题的业务域识别、参数抽取、结果展示、失败提示。
- 首页右上角展示运行可用状态。
- 运行页正式问数依赖默认 Agent 阻断项配置完整可用（配置生效），包括 LLM、嵌入模型、数据源连接和 Schema 初始化；Skill、知识、语义模型、预设问题和语音配置缺失不阻断文本问数。
- MVP 阶段不单独建设用户鉴权管理，入口按当前系统可访问范围开放。
- 语音输入属于 MVP 必做，前端上传 WAV 后由后端 ASR 代理返回识别文本并填入输入框，由用户确认后发送。

**不包含范围**

- MOM 主数据维护、工单/库存/设备原始业务单据编辑。
- 大屏报表、BI 自助建模、跨系统工作流审批。
- 复杂多轮任务执行，如自动创建工单、自动调整库存、安全库存策略写入。
- 智能体训练、模型微调、知识库运维后台的完整能力。
- 生产进度、生产达成率等非三类核心查询能力。

**相邻边界**

- MOM 智能问数负责问答入口、智能体编排、结果表达与会话留痕。
- MOM 业务系统负责提供标准数据、指标口径和查询能力。
- 智能体配置模块负责维护模型、数据源、知识、语义、Skill 和预设问题，运行页只消费配置完整可用的配置（配置生效）。
- 访问 API 为后置能力，MVP 阶段用户侧不维护 API Key、调用说明和外部接入能力。
- MVP 阶段不单独建设用户鉴权管理和用户登录注册管理。

**验收条件**

- 用户点击“新会话”后，主区进入空会话态，左侧新增或激活新会话记录。
- 用户可对单个历史会话修改标题，修改后左侧会话列表同步展示新标题。
- 用户可置顶单个历史会话，置顶会话展示在历史会话列表前部。
- 用户可删除单个历史会话，删除后该会话不再展示在历史会话列表中。
- 用户点击任一预设问题（推荐问题）卡片后，输入区或消息区触发对应问题查询，并返回结构化答案。
- 用户在问数处理中点击暂停会话按钮后，系统中止当前问数输出，保留当前会话和已产生内容。
- 用户输入“按工单状态统计当前工单数量”后，系统能识别为工单统计意图，并返回各状态数量。
- 用户输入“查询低于安全库存的物料”后，系统能识别为库存预警意图，并返回物料清单或无数据提示。
- 用户输入“分析设备停机时长”后，系统能识别为设备分析意图，并返回停机时长统计。
- 当业务结果为空、问题无法识别或配置不可用时，页面给出明确提示，且不生成误导性业务结论。
- 当默认 Agent 配置不完整或不可用时，运行页应提示配置不可用或服务不可用，不能继续输出业务结论。
- 用户可通过语音输入生成文本，确认发送后按文本问数链路返回结果或明确边界提示。

默认 Agent 可用性口径：

| 配置项 | 用户侧处理 |
|---|---|
| LLM 未配置或不可用 | 阻断正式问数，提示配置不可用。 |
| 嵌入模型未配置或不可用 | 阻断正式问数，提示配置不可用。 |
| 生效数据源不存在或连接失败 | 阻断正式问数，提示配置不可用。 |
| Schema 未初始化或初始化失败 | 阻断正式问数，提示配置不可用。 |
| 语义模型未就绪 | 不阻断问数，可提示结果准确性和指标口径存在风险。 |
| Skill 无可用配置 | 不阻断基础问数。 |
| 知识配置为空或未向量化 | 不阻断基础问数。 |
| 预设问题为空 | 不阻断手动输入，推荐问题区可为空。 |
| 语音配置停用或未配置 | 不阻断文本问数；用户点击语音识别时提示语音输入不可用。 |

### 2.3 功能描述

#### 2.3.1 会话管理

| 项目 | 说明 |
|---|---|
| 功能目标 | 支持用户创建、查看、切换和管理历史会话，保留问答上下文 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 用户可访问用户侧运行页 |
| 触发方式 | 点击“新会话”、点击左侧历史会话项，或点击单个会话的管理入口 |
| 系统处理逻辑 | 新建会话时初始化上下文；切换历史会话时加载会话消息、标题、创建/更新时间；修改标题、置顶、删除时更新对应会话状态 |
| 输出结果 | 主区展示当前会话内容或空会话预设问题；左侧历史会话列表展示最新标题、置顶状态和删除后的列表结果 |
| 异常处理 | 会话加载失败时提示重试；无历史会话时展示空列表说明；标题为空时禁止保存；删除会话前需提示确认 |
| 涉及页面 | 用户侧运行页左侧栏、主会话区 |

会话管理动作：

- 修改会话标题：用户可修改单个会话标题，标题为空时禁止保存。
- 置顶会话：用户可将常用会话置顶，置顶会话优先展示。
- 删除会话：用户可删除单个会话，删除前需二次确认，删除后不再展示在历史会话列表中。

验收标准：

- 用户点击“新会话”后，主区进入空会话态，左侧新增或激活新会话记录。
- 用户点击历史会话后，主区加载对应会话消息。
- 用户修改会话标题后，左侧会话列表同步展示新标题。
- 用户置顶会话后，该会话优先展示在历史会话列表前部。
- 用户删除会话前，系统提示二次确认；确认删除后，该会话不再展示在历史会话列表中。

#### 2.3.2 会话展示

| 项目 | 说明 |
|---|---|
| 功能目标 | 展示用户与默认 Agent 的问答过程，使用户能够在同一会话窗口中查看上下文、问数结果和异常提示 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 用户已进入有效会话 |
| 触发方式 | 用户发送文本问题、发送语音识别后的文本、点击预设问题或切换历史会话 |
| 系统处理逻辑 | 会话窗口按时间顺序展示用户消息、语音识别后的文本、Agent 回复、问数结果和系统提示 |
| 输出结果 | 展示文本答案、表格、指标卡、饼图、折线图、柱状图、简要分析结论、加载状态和异常提示 |
| 异常处理 | 问数处理中展示加载状态和暂停会话按钮；无数据、无法识别、配置不可用、服务异常时展示明确提示，不生成误导性业务结论 |
| 涉及页面 | 用户侧运行页主会话区 |

展示内容：

- 用户消息：展示用户输入的文本问题。
- 语音识别文本：语音输入后展示识别文本，用户确认后手动发送。
- Agent 回复：展示默认 Agent 返回的回答内容。
- 表格结果：用于展示明细清单类结果，如低于安全库存物料清单。
- 指标卡结果：用于展示关键数值，如工单总数、异常数量、停机总时长。
- 饼图结果：用于展示占比类结果，如工单状态占比、设备停机原因占比。
- 折线图结果：用于展示趋势类结果，如库存变化趋势、设备停机时长趋势。
- 柱状图结果：用于展示分类对比类结果，如不同设备停机时长对比、不同状态工单数量对比。
- 异常提示：用于展示无数据、无法识别、配置不可用、语音输入不可用等情况。
- 调试过程：MVP 用户侧默认不展示 SQL、Python 代码和原始 HTML；如后续开放调试信息，需另行增加权限控制、脱敏和安全渲染。

验收标准：

- 会话窗口可按顺序展示用户消息和 Agent 回复。
- 用户提交问题后，处理中状态有明确展示。
- 问数处理中可展示暂停会话按钮，用户点击后中止当前问数输出。
- 问数结果可按内容类型展示为文本、表格、指标卡、饼图、折线图或柱状图。
- 图表需包含标题、维度名称、指标值和必要单位。
- 无数据、无法识别、配置不可用或服务异常时，页面展示明确提示。
- 切换历史会话后，主会话区可展示该会话的历史问答内容。

#### 2.3.3 智能体配置入口

| 项目 | 说明 |
|---|---|
| 功能目标 | 提供进入管理侧智能体配置页面的入口，用于跳转查看和维护默认 Agent 的模型、数据源、知识、语义、Skill 和预设问题 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 页面可访问；管理侧智能体配置页面可访问 |
| 触发方式 | 点击“智能体配置”按钮 |
| 系统处理逻辑 | 跳转进入管理侧智能体配置页面；MVP 阶段用户侧不做额外鉴权判断；页面不可用时提示稍后重试 |
| 输出结果 | 打开管理侧智能体配置页或展示不可用提示 |
| 异常处理 | 配置服务不可用时提示稍后重试 |
| 涉及页面 | 用户侧运行页左侧栏、智能体配置页 |

验收标准：

- 所有可访问用户均可从左侧栏进入智能体配置页。
- 点击入口后可打开管理侧智能体配置页面。
- 配置页不可用时，页面展示明确提示。
- MVP 阶段不单独建设用户鉴权管理和用户登录注册管理。

#### 2.3.4 预设问题快捷查询

| 项目 | 说明 |
|---|---|
| 功能目标 | 降低用户提问成本，提供高频 MOM 查询入口，预设问题在用户侧可展示为推荐问题 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 预设问题已配置 |
| 触发方式 | 点击工单、库存、设备预设问题（推荐问题）卡片 |
| 系统处理逻辑 | 将预设问题写入当前会话，并发起问数 |
| 输出结果 | 返回对应统计、清单或分析答案 |
| 异常处理 | 用户无结果、无数据时返回明确说明 |
| 涉及页面 | 用户侧运行页预设问题区、会话消息区 |

验收标准：

- 首页可展示已启用的预设问题。
- 用户点击预设问题后，系统将该问题写入当前会话。
- 点击预设问题后可触发默认 Agent 问数链路。
- 预设问题返回结果后，结果展示在当前会话窗口中。
- 无数据或无法返回结果时，页面展示明确提示。

#### 2.3.5 自然语言问数

| 项目 | 说明 |
|---|---|
| 功能目标 | 支持用户以自然语言查询 MOM 业务问题 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 用户已可访问用户侧运行页；问数能力已生效 |
| 触发方式 | 输入文本后点击发送；或通过语音入口输入后发送；问数处理中点击暂停会话按钮 |
| 系统处理逻辑 | 发送后识别业务域与意图，抽取时间、状态、物料、设备等参数，组织答案；暂停时中止当前流式问数任务 |
| 输出结果 | 文本、表格、指标卡、简要分析结论或当前问数已暂停提示 |
| 异常处理 | 无法识别意图时引导用户补充条件；参数缺失时发起澄清；语音输入不可用时提示用户改用文本输入 |
| 涉及页面 | 用户侧运行页主会话区、底部输入区 |

验收标准：

- 用户可在输入区输入文本问题并发送。
- 用户可在问数处理中点击暂停会话按钮，中止当前问数输出。
- 系统可识别工单、库存、设备三类基础问数意图。
- 系统可对缺少必要条件的问题给出澄清提示。
- 问数结果可展示为文本、表格、指标卡或图表。
- 问题无法识别、业务结果为空或配置不可用时，页面展示明确提示。

#### 2.3.6 语音问数

| 项目 | 说明 |
|---|---|
| 功能目标 | 支持用户通过语音识别降低输入成本 |
| 适用角色 | 所有可访问用户 |
| 前置条件 | 用户已可访问用户侧运行页；语音输入配置已启用；默认 Agent 问数能力已配置完整可用（配置生效） |
| 触发方式 | 点击输入区语音按钮，录入语音问题，前端上传 WAV 后端识别并填入输入框，用户手动发送 |
| 系统处理逻辑 | 前端采集音频并转换为 WAV 后上传后端，后端代理调用 ASR 并返回识别文本；用户确认发送后，后端按普通文本问题进入默认 Agent 问数链路 |
| 输出结果 | 展示转写后的文本；用户发送后返回问数结果、无数据提示、澄清提示或边界提示 |
| 异常处理 | 语音为空、格式不支持、文件超限、识别文本为空、ASR 调用失败、调用超时、调用频率超限或语音输入配置不可用时，提示用户重新录入或改用文本输入 |
| 涉及页面 | 用户侧运行页主会话区、底部输入区 |

验收标准：

- 用户可通过语音入口录入问题。
- 前端上传 WAV 后，后端可将语音内容识别成文本并返回输入框。
- 识别文本不会自动提交，用户确认后手动发送。
- 用户发送后，识别文本按普通文本问题进入默认 Agent 问数链路。
- 语音输入不可用时，页面展示明确提示，文本问数仍可继续使用。

接口约束：

- MVP 语音输入仅支持前端上传 WAV，浏览器原始录音如为 WebM/OGG，由前端转换为 WAV 后再上传。
- 后端按语音配置校验文件大小、音频时长、MIME 类型、WAV 文件头、ASR 调用超时和调用频率。
- 语音识别接口只返回识别文本，不创建会话消息，不自动发起问数。
- 语音输入失败、超限、超时、频率受限或识别为空时，不进入问数链路。

| 类别 | 名称 | 载体 | 用途 | 是否必需 | 说明 |

| 展示配置 | 预设问题条目 | 管理侧配置对象 | 支撑用户侧推荐问题展示 | 是 | 用户侧只展示启用状态的预设问题。 |

| 结果数据 | 结果 Payload | 会话消息扩展数据 | 支撑问数结果摘要、状态、类型和结构化展示 | 是 | 随 Agent 回复消息保存到 metadata.resultPayload。 |
| 展示数据 | 表格结果结构 | resultPayload 子结构 | 支撑明细清单类结果展示 | 是 | 如低于安全库存物料清单。 |
| 展示数据 | 指标卡结果结构 | resultPayload 子结构 | 支撑关键数值展示 | 是 | 如工单总数、停机总时长。 |
| 展示数据 | 图表结果结构 | resultPayload 子结构 | 支撑饼图、折线图、柱状图展示 | 是 | 图表需包含标题、维度、指标和必要单位。 |
| 提示数据 | 运行提示结构 | resultPayload 子结构 | 支撑无数据、无法识别、配置不可用、语音输入不可用等提示 | 是 | 提示内容不得生成误导性业务结论。 |

说明：

- 用户侧不维护模型、数据源、知识、语义模型、Skill 和访问 API 配置，只消费管理侧已配置完整可用的默认 Agent 能力（配置生效）。
- 访问 API 为后置能力，MVP 阶段用户侧不维护 API Key、调用说明和外部接入能力。
- 预设问题为正式配置对象名称，用户侧展示时可称为推荐问题。
- 语音输入由前端上传 WAV、后端 ASR 代理识别成文本并填入输入框，用户发送后按文本消息保存；发送后的消息 `inputType=VOICE`。
- 最终结果随 Agent 回复消息保存到 `metadata.resultPayload`，不单独建设结果中心。
- 图表结果至少支持饼图、折线图和柱状图。
