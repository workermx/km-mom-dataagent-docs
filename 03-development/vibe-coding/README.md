## 规则

- 同一时间，同一开发部分只允许一份活动计划。
- 新任务开始前，先确认当前活动计划是否已经存在。
- 活动计划完成后，再创建下一份。
- 支持件只提供格式、检查项和提示，不单独指挥开发。
