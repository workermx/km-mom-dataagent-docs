## 怎么用

如果要开始一个新任务，先复制 `任务卡模板.md`，填清楚目标、范围、验收和涉及文件。

如果只是想继续当前项目进度，直接说：

```text
按 Vibe Coding 手册继续执行当前任务。
```

如果要进入后端开发，直接说：

```text
按后端落地检查表，开始设计 FastAPI + LangChain 的接口和服务结构。
```
